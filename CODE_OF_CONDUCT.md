# Code of Conduct

This project follows simple norms:

1. **Be kind.** Disagree on ideas, not on people.
2. **Assume good intent.** Most heat in OSS issues comes from misread tone, not bad actors.
3. **Stay on scope.** This repo is about an AI agent — off-topic debate doesn't belong here.
4. **Credit prior work.** When you port code from another OSS project, link to it and follow its license.
5. **No spam.** Includes spam PRs, spam issues, and astroturfed contributions.

Maintainers may close issues or PRs that violate these norms. Repeated violations may result in a block.

For private concerns, email `support@meshpilot.app`.
